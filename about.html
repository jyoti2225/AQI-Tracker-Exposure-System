AQI Exposure Monitoring System

This is an AQI Exposure Monitoring System. It is built using Flask, Oracle Database, HTML, CSS, and JavaScript. It shows AQI, temperature, and rain data for different cities. It also provides health advisory based on AQI levels. Users can select cities and view environmental conditions. The system helps people make safe outdoor decisions.
